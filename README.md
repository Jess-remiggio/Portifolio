Este projeto foi desenvolvido como parte da minha evolução no desenvolvimento front-end, com o objetivo de criar um portfólio pessoal simples, organizado e visualmente agradável, utilizando apenas HTML e CSS. A proposta foi construir uma página que apresentasse minhas informações de forma clara, com seções bem definidas, imagens, ícones e um layout estruturado, permitindo exercitar conceitos fundamentais de construção de interfaces e identidade visual. Durante o desenvolvimento, trabalhei a organização dos elementos na página, a criação de áreas específicas para apresentação pessoal, projetos e contatos, além da aplicação de estilos que deixassem a interface moderna e coerente com a proposta de um portfólio digital.

Ao longo do processo, pude reforçar a importância da hierarquia visual, do uso adequado de cores, tipografia e espaçamentos, além da responsividade básica para garantir que o conteúdo permanecesse legível em diferentes tamanhos de tela. Também pratiquei a separação entre estrutura e estilo, mantendo o HTML focado no conteúdo e o CSS responsável pela aparência, o que contribuiu para um código mais limpo e organizado. A utilização de imagens e ícones ajudou a tornar o portfólio mais dinâmico e visualmente interessante, reforçando a identidade do projeto.

Este portfólio representa um passo importante na minha trajetória como desenvolvedora, pois me permitiu aplicar na prática conceitos essenciais de front-end, explorar a criação de layouts e desenvolver uma interface funcional que apresenta quem eu sou e o que estou construindo. Além disso, reforçou minha capacidade de transformar ideias em páginas reais, utilizando apenas as tecnologias fundamentais da web e aprimorando minha autonomia no processo de criação.
